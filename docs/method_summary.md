# 已实现方法简要说明

## 1. 时间对齐

磁强计时间为北京时间系统时间。NovAtel GPGGA 中的时间字段按 UTC 处理，转为北京时间：

```text
t_BJ = t_GPGGA_UTC + 8 h
```

当前 GPGGA 流程默认不减 18 s 闰秒。如果后续使用 GPS 周秒或 NovAtel 原生日志头时间，再单独考虑 GPS time 到 UTC 的闰秒转换。

## 2. 轨道坐标与 0 点

所有有效 SPAN 坐标先转换为局部 East-North 平面坐标，再用 PCA 拟合一条共享轨道轴。4.14 和 5.13 共用同一条轨道轴和同一个物理 0 点，地图距离为：

```text
s = dot(p - p0, u)
```

其中 `p` 是局部平面坐标，`p0` 是轨道 0 点，`u` 是轨道方向单位向量。输出磁图以 0.5 m 为间隔。

## 3. 往返方向与三轴磁场

小车往返时车头方向变化，磁传感器固定在车体上，因此 raw `x/y/z` 不能直接跨趟平均。当前处理逻辑为：

1. 每个连续观测段单独估计车体系磁场中位数；
2. 对 `x/y/z` 去除该段中位数，得到车体磁异常；
3. 根据沿轨道距离增减判断运动方向；
4. 把车体磁异常旋转到统一轨道坐标系。

由于不同日期三轴幅值仍存在明显差异，当前最可靠的主匹配特征仍是 total 场，`Y anomaly` 可作为辅助特征。

## 4. 磁图融合

每个 0.5 m 地图点可能有多趟观测。融合磁图默认使用稳健中位数：

```text
m_map(s) = median(m_1(s), m_2(s), ..., m_N(s))
```

同时保留均值、标准差和 MAD：

```text
MAD(s) = median(|m_i(s) - median(m_i(s))|)
```

中位数比均值更能抵抗掉头、异常尖峰和单趟局部干扰。

## 5. 相似度指标

跨日期磁图和不同趟数据主要使用 Pearson 相关系数评价形状相似性：

```text
r = sum((x_i - mean(x)) (y_i - mean(y))) /
    sqrt(sum((x_i - mean(x))^2) sum((y_i - mean(y))^2))
```

此外还计算梯度相关和去偏置 RMSE，用于区分“整体幅值不同但形状相似”和“局部特征也不一致”的情况。

## 6. 初步匹配 baseline

当前 baseline 是滑窗归一化互相关 NCC。给定 query 磁曲线 `q` 和参考磁图窗口 `m_k`，先分别 z-score 标准化，再计算：

```text
NCC(k) = mean(q'_i m'_{k+i})
```

得分最高的候选窗口起点作为预测位置。该方法简单、可解释，但容易被相似片段干扰，因此后续需要加入运动连续性约束、方向自标定和 Graph SLAM。
